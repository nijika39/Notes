# 图

本章介绍了图的各种容易遗忘的名词定义、图的性质等，常在选择题中考察。

---

## 一、 图的概念与性质

### 1. 顶点的度
* **[[无向图]]**：所有顶点的度之和为边数 $2$ 倍。
* **[[有向图]]**：所有顶点的[[入度]]之和等于[[出度]]之和。

### 2. 路径与回路
* **[[简单路径]]**：路径序列中**顶点不重复出现**的路径。
* **[[简单回路]]**：除第一个顶点和最后一个顶点外，**其余顶点不重复出现**的回路。
* 各顶点的度均大于等于 $2$ 的无向图必有回路

### 3. [[子图]]
> [!warning] 易错点
> 假设存在图 $G = (V, E)$，**并非** $V$ 和 $E$ 的任意子集都能构成 $G$ 的子图。
> * 原因：子图必须是一个“合法”的图，即子图中边集 $E'$ 所关联的顶点必须全部包含在子顶集 $V'$ 中。

### 4. [[连通分量]]
* 无向图中的**极大连通子图**称为连通分量。
* > [!note] 注意
  > [[生成树]]是包含所有顶点的**极小连通子图**，因此生成树通常**不是**连通分量（除非原图本身就是一棵树）。

### 5. 极端图结构的边数计算（高频选择题考点）
* **非连通无向图**：假设一个图有 $n$ 个顶点，若该图为无向图且**非连通**，其拥有最大边数的情形为：$n-1$ 个顶点构成一个[[完全图]]，剩下 $1$ 个顶点孤立。最大边数为：
  $$\frac{(n-1)(n-2)}{2}$$
* **强连通有向图**：若有向图为[[强连通图]]，其拥有最少边数的情形为：所有顶点构成一个环。最小边数为：
  $$n$$

### 6. [[生成树]]
* 连通图的生成树是一个包含所有顶点的**极小连通子图**。
* **特征**：
  * 增加任何一条边都会形成一个回路。
  * 移除任何一条边都会变成非连通图。

## 二、 图的存储

### 1. [[邻接表]]与[[邻接矩阵]]时间复杂度对比

对于含有 $|V|$ 个顶点、 $|E|$ 条边的图，计算特定顶点或所有顶点的度、入度、出度时的时间复杂度如下：

| 求解目标 | 邻接矩阵 | 邻接表 | 核心原因 |
| :--- | :--- | :--- | :--- |
| **求某顶点的出度** | $O(\|V\|)$ | $O(\text{out-degree}) \le O(\|V\|)$ | 邻接矩阵要扫描一整行；邻接表只需遍历该顶点自己的单链表。 |
| **求某顶点的入度** | $O(\|V\|)$ | $O(\|V\| + \|E\|)$ | 邻接矩阵扫描一整列；邻接表必须扫描全图所有的边。 |
| **求所有顶点的入度** | $O(\|V\|^2)$ | $O(\|V\| + \|E\|)$ | 邻接矩阵需扫描整个二维数组；邻接表通扫一遍全图（配辅助数组）即可。 |

### 2. 度与邻接矩阵的对应关系
* **无向图**：邻接矩阵一定是对称矩阵，顶点 $i$ 的 $\text{度} = \text{出度} = \text{入度} =$ 邻接矩阵中第 $i$ 行（或第 $i$ 列）中不为 $0$ 且不为 $\infty$ 的元素个数。
* **有向图**：
  * 顶点 $i$ 的**出度** = 邻接矩阵中**第 $i$ 行**中不为 $0$ 且不为 $\infty$ 的元素个数。
  * 顶点 $i$ 的**入度** = 邻接矩阵中**第 $i$ 列**中不为 $0$ 且不为 $\infty$ 的元素个数。

### 3. 邻接矩阵幂的物理意义
设图 $G$ 的邻接矩阵为 $A$，则 $A$ 的 $n$ 次方中元素 $A^n[i][j]$ 代表：**从顶点 $i$ 到顶点 $j$ 长度为 $n$ 的路径数量**。

### 4. 表示的唯一性
* 对于同一个图，在顶点编号固定的情况下：
  * **邻接矩阵**的表示**唯一**。
  * **邻接表**的表示**不唯一**（取决于建立邻接表的算法和边的输入次序）。

### 5. 高级存储结构对比
* **[[十字链表]]**：只适用于**有向图**，顶点表使用顺序存储，对同一个图的表示**不唯一**。
* **[[邻接多重表]]**：只适用于**无向图**，顶点表使用顺序存储，对同一个图的表示**不唯一**。
* 在邻接多重表中，顶点 $i$ 的度等于表的边结点中出现 $i$ 的次数。

## 三、 图的遍历

### 1. BFS 求解单源最短路径
* **[[BFS]]（广度优先搜索）** 可用于求解**非带权图**（或权值相等图）的[[单源最短路径]]问题。

### 2. 遍历生成树的唯一性
* 对于同一个图，在顶点编号固定且从同一源点出发的情况下：
  * 若用**邻接矩阵**存储：得到的[[生成树]]是**唯一**的。
  * 若用**邻接表**存储：得到的生成树是**不唯一**的。

### 3. DFS 和 BFS 的时空复杂度对比

进行全图遍历时，两种算法的复杂度如下：

| 遍历方式 | 邻接表存储 (时间复杂度) | 邻接矩阵存储 (时间复杂度) | 空间复杂度 |
| :--- | :--- | :--- | :--- |
| **[[DFS]]** | $O(\|V\| + \|E\|)$ | $O(\|V\|^2)$ | $O(\|V\|)$ （最坏情况下递归栈深度） |
| **[[BFS]]** | $O(\|V\| + \|E\|)$ | $O(\|V\|^2)$ | $O(\|V\|)$ （辅助队列和标记数组的大小） |

### 4. 遍历与连通分量的关系
* 遍历无向图时，在 `DFSTraverse()` 或 `BFSTraverse()` 中，调用 `DFS()` 或 `BFS()` 的次数，**恰好等于该图的连通分量个数**。
