# 栈和队列的应用

栈和队列在实际计算中有广泛的应用。其中，栈最经典的两个应用是 **[[括号匹配]]** 和 **[[表达式求值]]**。

---

## 1. 括号匹配

* **一句话定义**：利用栈“后进先出”的特性，来检测括号嵌套的合法性。
*(此处略，详情可参见对应算法专题笔记。)*

---

## 2. 表达式

### 2.1 基础定义
* **前缀表达式（波兰式）**：运算符在操作数之前。
* **后缀表达式（逆波兰式）**：运算符在操作数之后。
* **中缀表达式**：运算符在操作数之间（我们日常使用的数学表达式）。

---

### 2.2 中缀表达式转后缀表达式

#### A. 手算转换方式（遵循 “左优先” 原则）
1. **加括号**：按运算优先级对整个表达式逐层加括号。
2. **移符号**：将每个运算符移动到其所在括号的**右括号后面**。
3. **去括号**：删除所有括号。

> 📝 **示例**：
> * **原式**：$A + B * (C - D) - E / F$
> * **第一步（逐层加括号）**：$((A + (B * (C - D))) - (E / F))$
> * **第二步（运算符移至右括号后）**：$((A \ (B \ (C \ D) - ) * ) + \ (E \ F) / ) -$
> * **第三步（去括号）**：$A \ B \ C \ D - * + E \ F / -$

#### B. 计算机编程方式（机算）
* **基本步骤**（从**左向右**扫描中缀表达式）：
  1. **遇到操作数**：直接加入后缀表达式。
  2. **遇到界限符**：若为 `(`，直接入栈；若为 `)`，不入栈，且不断弹出栈顶运算符并加入后缀表达式，直到遇到 `(`，将其弹出并丢弃。
  3. **遇到运算符**：
     * 若其优先级**高于**栈顶运算符，或栈顶为 `(`，则直接入栈。
     * 若其优先级**低于或等于**栈顶运算符，则依次弹出栈中运算符并加入后缀表达式，直到栈空，或栈顶为 `(`，或遇到优先级更低的运算符为止，再将其入栈。
  4. **扫描完毕**：扫描完所有字符后，将栈中剩余运算符依次弹出并加入后缀表达式。

#### C. C 语言核心算法实现代码
```c
#include <stdio.h>

#define MAX_SIZE 100

// 定义顺序栈结构
typedef struct {
    char data[MAX_SIZE];
    int top;
} SqStack;

// 获取运算符优先级
int getPriority(char op) {
    if (op == '+' || op == '-') return 1;
    if (op == '*' || op == '/') return 2;
    if (op == '^') return 3;
    return 0; // 括号或非运算符
}

// 中缀转后缀算法函数（假设输入字符数组 infix，输出存储在 postfix 中）
void infixToPostfix(char infix[], char postfix[]) {
    SqStack s;
    InitStack(&s);
    int i = 0, j = 0;
    char c, topOp;

    while (infix[i] != '\0') {
        c = infix[i];

        // 1. 遇到操作数（简写，假设为单字符如 'A'-'Z', '0'-'9'）
        if ((c >= 'a' && c <= 'z') || (c >= 'A' && c <= 'Z') || (c >= '0' && c <= '9')) {
            postfix[j++] = c;
        }
        // 2. 遇到界限符
        else if (c == '(') {
            Push(&s, c);
        }
        else if (c == ')') {
            while (!StackEmpty(s) && GetTop(s, &topOp) && topOp != '(') {
                Pop(&s, &topOp);
                postfix[j++] = topOp;
            }
            Pop(&s, &topOp); // 弹出 '(' 并丢弃
        }
        // 3. 遇到运算符
        else if (c == '+' || c == '-' || c == '*' || c == '/' || c == '^') {
            // 当栈不空、栈顶不是'('，且栈顶运算符优先级【大于或等于】当前运算符时，不断出栈
            while (!StackEmpty(s) && GetTop(s, &topOp) && topOp != '(' && getPriority(topOp) >= getPriority(c)) {
                Pop(&s, &topOp);
                postfix[j++] = topOp;
            }
            Push(&s, c); // 当前运算符入栈
        }
        i++;
    }

    // 4. 将栈中剩余运算符弹出
    while (!StackEmpty(s)) {
        Pop(&s, &topOp);
        postfix[j++] = topOp;
    }
    postfix[j] = '\0'; // 加上字符串结束符
}
```

### 2.3 后缀表达式求值（求值）

* **运行步骤**（从**左至右**依次扫描后缀表达式）：
  1. 若当前项为**操作数**：压入栈中。
  2. 若当前项为**运算符** `<op>`：从栈中弹出两个数。
     * **先弹出**的为**右操作数 $b$**。
     * **后弹出**的为**左操作数 $a$**。
     * 执行运算 $a \ \langle op \rangle \ b$，然后将计算结果压入栈中。

---

### 2.4 中缀表达式转前缀表达式

#### A. 手算转换方式（遵循 “右优先” 原则）
* **步骤**：步骤 1（加括号）和 3（去括号）与中缀转后缀完全一致；步骤 2 改为 **将运算符移动到对应左括号的左侧**。

#### B. 计算机编程方式（机算）
* **基本步骤**（从**右向左**扫描中缀表达式）：
  1. **遇到操作数**：直接加入前缀表达式。
  2. **遇到界限符**：若为 `)`，直接入栈；若为 `(`，则不断弹出栈顶运算符并加入前缀表达式，直到遇到 `)`，将其弹出并丢弃。
  3. **遇到运算符**：
     * 若其优先级**高于或等于**栈顶运算符，或栈顶为 `)`，则直接入栈。
     * 若其优先级**低于**栈顶运算符，则不断弹出栈顶运算符并加入前缀表达式，直至栈空，或栈顶为 `)`，或遇到优先级比其更高或同级的运算符为止，再将当前运算符入栈。
  4. **扫描完毕**：扫描完所有字符后，将栈中剩余运算符弹出并加入前缀表达式。
  5. **逆序**：将得到的表达式**反转（逆序）**，即得到最终的前缀表达式。

---

### 2.5 前缀表达式求值（求值）

* **运行步骤**（从**右至左**依次扫描前缀表达式）：
  1. 若当前项为**操作数**：直接压入栈中。
  2. 若当前项为**运算符** `<op>`：从栈中弹出两个数。
     * **先弹出**的为**左操作数 $a$**。
     * **后弹出**的为**右操作数 $b$**。
     * 执行运算 $a \ \langle op \rangle \ b$，然后将计算结果压入栈中。

---

## 🔑 核心考点：手算与机算一致性原则

由于在转换时算术序列可能不唯一，为保证**手算结果与机算结果完全一致**，必须严格遵循以下原则：

| 转换目标 | 核心原则 | 口诀（三要素） | 详细说明 |
| :--- | :--- | :--- | :--- |
| **中缀转后缀** / **后缀求值** | **左优先** | **“两左一后”** | ① 遵循**左优先**原则（手算时同级运算符先算左侧）<br>② 机算时从**左**往右扫描<br>③ 运算符放在操作数之**后** |
| **中缀转前缀** / **前缀求值** | **右优先** | **“两右一前”** | ① 遵循**右优先**原则（手算时同级运算符先算右侧）<br>② 机算时从**右**往左扫描<br>③ 运算符放在操作数之**前** |
